\# OpenPose Pretrained Models (Mirror)



This is a mirror of the pretrained `.caffemodel` weights required to run \[CMU OpenPose](https://github.com/CMU-Perceptual-Computing-Lab/openpose).  

The original server at CMU (`posefs1.perception.cs.cmu.edu`) is offline, so this HuggingFace repo hosts the full set for convenience.



\## 📂 Models Included

\- Body Pose (BODY\_25): `models/pose/body\_25/pose\_iter\_584000.caffemodel`

\- Body Pose (COCO): `models/pose/coco/pose\_iter\_440000.caffemodel`

\- Body Pose (MPI): `models/pose/mpi/pose\_iter\_160000.caffemodel`

\- Face: `models/face/pose\_iter\_116000.caffemodel`

\- Hand: `models/hand/pose\_iter\_102000.caffemodel`
